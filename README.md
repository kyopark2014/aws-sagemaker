# AWS SageMaker


## Basic

SageMaker는 완전 관리형 머신 러닝 학습 서비스로서, 데이터 과학자가 빠르고 쉽게 모델 개발 및 학습을 할 수 있도록 지원합니다. 


## SageMaker Training

SageMaker에서 제공되는 jupyter Notebook을 통해, 모델을 개발할 수 있습니다. 하지만, 노트북 인스턴스에서 모델 학습을 수행할 수 있지만, 더 높은 성능의 CPU를 요구할때 노트북 인스턴스를 Scale-Up 하는것은 비용적으로 효율적이지 않습니다. 따라서, 별도 인스턴스를 띄워서 모델 학습을 진행하는데 이것을 SageMaker Training이라고 합니다.
